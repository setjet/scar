# Assets

Put your demo video here so it can be referenced from the README.

## Demo video

1. Add your file at: `assets/demo.mp4`
2. The root `README.md` embeds it via:

```html
<video src="assets/demo.mp4" controls muted playsinline style="max-width: 100%;"></video>
```

Notes:
- Keep the MP4 reasonably small so GitHub can render it smoothly.
- If it’s large (near/over ~100MB), use Git LFS instead.
